Sticky elements

Put [sticky (or sticky-top), sticky-right, sticky-bottom, sticky-left] class in header and/or navbar element, but it also works with other elements.

Put [sticky-only-pc] if you want to block mobile sticky. Put [sticky-only-mobile] if you want to block pc sticky.

Code

<nav class="sticky">...</nav>

Or

<div class="nav sticky">...</nav>

Result
